# Optimization of Emergency Evacuation Routes in Calumpit

## Overview
This repository contains our exposition paper on the optimization of emergency evacuation routes in Barangay Corazon, Calumpit, Bulacan using graph theory, network flow models, and linear programming.

The study analyzes how evacuation efficiency is affected by transportation limitations and flood conditions, and determines the maximum evacuation flow using mathematical modeling.

## Objectives
- Determine evacuation demand in Barangay Corazon  
- Evaluate transportation capacity under flood conditions  
- Analyze evacuation routes and identify bottlenecks  
- Apply Max-Flow Min-Cut Theorem in real-life scenarios  

## Methods Used
- Graph Theory  
- Network Flow Models  
- Linear Programming  
- Ford-Fulkerson Algorithm  
- Shortest Path Analysis  

## Key Findings
- Maximum evacuation flow: **16–19 persons per cycle**  
- Required evacuation trips: **240–285 trips**  
- Estimated evacuation time: **32–38 hours**  
- Main bottleneck: **limited transportation capacity**, not road network  

## Scenario Insight
Adding one more transport vehicle significantly improves evacuation time (down to ~25–30 hours), showing that resource allocation is critical in disaster planning.
